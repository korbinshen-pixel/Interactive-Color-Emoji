# Interactive Color Emoji

一个可交互的SVG表情变色工具，适用于英语教学。

## 功能特点
- ✅ 5个表情可选
- ✅ 8种颜色可选
- ✅ 响应式设计，支持手机和电脑
- ✅ 点击颜色即可改变表情颜色
